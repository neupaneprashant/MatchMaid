# Routes Overview

| Route                      | Description                            | Access Role     |
|---------------------------|----------------------------------------|-----------------|
| `/`                       | Landing page with role selection       | All users       |
| `/signup`                | Sign up or log in                      | All users       |
| `/home`                  | Main swipe interface                   | Hirees          |
| `/chat`                  | General chat window                    | Authenticated   |
| `/tinder-shuffle/review` | View reviews                           | Hirees          |
| `/maid-portal`           | Maid dashboard                         | Maids only      |
| `/maid-schedule`         | Schedule availability management       | Maids only      |
| `/maid-chat`             | Dedicated maid chat                    | Maids only      |
| `/products`              | Product info or upsell page            | All users       |
| `/learn`                 | Education about hiring maids           | All users       |
| `/safety`                | Safety guidelines or tips              | All users       |
